Detailed Description

Definición de la clase Archer, un jugador especializado en ataques a distancia.

La clase Archer hereda de Player y redefine el comportamiento de ataque para reflejar el uso de habilidades y armas a distancia (arcos, ballestas, etc.).

El arquero tiene una probabilidad del 50% de asestar un "headshot" (impacto en la cabeza) que inflige el doble de daño. La implementación de attack() debe incorporar esa mecánica: calcular el daño base y, con 50% de probabilidad, multiplicarlo por 2.

Ejemplo de uso:

Archer ar("Legolas");
int dano = ar.attack();
// Si se produjo headshot, daño ya será el doble del valor base
std::cout << "Daño de arquero: " << dano << std::endl;

Member Function Documentation

◆ attack()

int Archer::attack ( )
overridevirtual

Implementación específica del ataque para arqueros.

Este método sobrescribe Player::attack() para calcular daño usando estadísticas relevantes (por ejemplo dexterity) y el arma equipada. Además, tiene una mecánica de "headshot":

  • 50% de probabilidad de asestar un headshot que causa daño doble.

La convención de retorno es simplemente el daño calculado (ya duplicado si hubo headshot). Si se desea distinguir el caso de headshot, se puede comprobar el daño respecto al valor base conocido o ampliar la API.

Ejemplo:

Archer a("Flecha");
int daño = a.attack();
// Supongamos que el daño base esperado era 6
if (daño == 12) // Detectamos headshot por duplicado
std::cout << "Headshot! Daño doble: " << daño << std::endl;
else
std::cout << "Impacto normal: " << daño << std::endl;
// Aplicar daño a un objetivo: target.takeDamage(daño, &a);
Returns
int Daño calculado por el ataque del arquero (doble si headshot).

Detailed Description

Clase base que modela a un jugador en el juego.

Player encapsula nombre, clase, puntos de vida, arma y estadísticas. Proporciona métodos para atacar, defender y recibir daño. Está diseñada para ser heredada por clases concretas (Warrior, Archer, Paladin, Mage) que implementen comportamiento específico.

Contrato resumido:

  • Entradas: operaciones de ataque/defensa y llamadas externas a setters.
  • Salidas: cambios en healthPoints, valores retornados por attack() y defend().
  • Errores: no lanza excepciones en la interfaz pública; asume punteros válidos para Weapon*.

Ejemplo de uso:

// Crear jugador sin arma inicialmente
Player p("Aldo", warrior);
// Asignar stats: strength, intelligence, dexterity, physicalDefence, magicDefence
int s[5] = {15, 3, 8, 5, 2};
p.setStats(s, 5);
// Asignar arma (suponiendo existencia de clase Weapon y constructor)
Weapon* sword = new Weapon("Espada corta", 5);
p.setWeapon(sword);
// Realizar ataque
int dmg = p.attack();
// Defender (ejemplo)
defendResult res = p.defend();
// Aplicar daño recibido por otro jugador
p.takeDamage(10, nullptr);

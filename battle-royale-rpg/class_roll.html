Detailed Description

Funciones utilitarias estáticas para generar tiradas aleatorias (dados).

Esta cabecera expone la clase Roll con métodos estáticos para simular tiradas comunes de dados usados en el juego (D100, D20, D10, D6, D2).

Todas las funciones devuelven un entero entre 1 y N (inclusive), donde N es la cantidad de caras del dado correspondiente.

Ejemplo de uso:

// Tirar un dado de 20 caras
int resultado = Roll::rollD20();
std::cout << "Tirada D20: " << resultado << std::endl;
// Comprobar crítico
if (resultado == 20) // crítico
std::cout << "¡Impacto crítico!" << std::endl;

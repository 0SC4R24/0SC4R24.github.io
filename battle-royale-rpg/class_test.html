Detailed Description

Utilidad para ejecutar pruebas y escenarios de combate entre jugadores.

Clase utilizada para realizar pruebas de combate entre jugadores.

La clase Test facilita ejecutar combates aleatorios y casos concretos entre dos instancias de Player. Está pensada para pruebas manuales o automatizadas durante el desarrollo.

Ejemplo de uso:

Player* p1 = new Player("A", warrior);
Player* p2 = new Player("B", mage);
Test t(p1, p2);
t.testRandomCombat();

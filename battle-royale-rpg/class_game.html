Detailed Description

Controla la lógica principal del juego: creación de jugadores y ejecución de turnos.

La clase Game encapsula la lista de jugadores, la inicialización de éstos y la secuencia de turnos entre atacantes y defensores. Diseñada para ser una clase simple que orquesta la simulación de un enfrentamiento entre varios Player.

Ejemplos de uso:

// Crear un juego para 4 jugadores, inicializarlos y jugar
Game* g = new Game(4);
g->initializePlayers();
g->playGame();
// Alternativamente, en código de prueba se puede invocar directamente los turnos
// (suponiendo acceso y contratos adecuados):
// g.turn(0, 1); // jugador 0 ataca a jugador 1

Detailed Description

Clase que gestiona datos compartidos y generadores aleatorios usados por el juego.

DataBase agrupa nombres y generadores para estadísticas, puntos de vida y armas. Todos los miembros y métodos son estáticos para poder acceder a ellos sin instanciar la clase.

Ejemplos de uso:

// Inicializar la base de datos (mostrar información de depuración)
DataBase::initializeDatabase(true);
// Obtener estadísticas aleatorias (para la clase por defecto / aleatoria)
int* stats = DataBase::getRandomStats();
// stats apunta a un array de tamaño DataBase::statsNames.size() (consultar implementación)
// Obtener estadísticas aleatorias para una clase concreta
// Ajusta el identificador de la clase según el enum definido en Player.h
playerClass cls = playerClass::Paladin;
int* paladinStats = DataBase::getRandomStats(cls);
// Obtener puntos de vida entre un rango
int hp = DataBase::getRandomHP(30, 60);
// Obtener puntos de vida para una clase
int mageHp = DataBase::getRandomHP(playerClass::Mage);
// Obtener un arma aleatoria
Weapon* w = DataBase::getRandomWeapon();
// No se asume aquí la semántica de propiedad del puntero; revisar la implementación
// para saber si hay que liberar el Weapon o si es administrado internamente.

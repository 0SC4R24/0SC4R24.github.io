Detailed Description

Definición de la clase Warrior (guerrero) — tanque que inflige más daño a baja vida.

Según el README del proyecto, el Warrior obtiene un aumento de daño cuando sus puntos de vida están por debajo del 50% de su máximo. Esta cabecera declara la clase Warrior, que hereda de Player, y documenta su comportamiento específico.

Ejemplo de uso:

Warrior w("Conan");
// Suponiendo que w.setHealthPoints() y w.setStats() ya fueron usados para configurar vida y stats
int dmg = w.attack();
std::cout << "Daño de guerrero: " << dmg << std::endl;

int Warrior::attack ( )
overridevirtual

Implementación de ataque del guerrero.

Sobrescribe Player::attack() y aplica la mecánica descrita en el README: si los puntos de vida actuales del guerrero están por debajo del 50% de maxHP, el daño calculado se ve incrementado (buff de furia).

Returns
int Daño final calculado por el ataque, ya con el buff aplicado si procede.

Ejemplo:

// Forzar estado de baja vida para ver el buff
w.setHealthPoints(w.getHealthPoints() / 4);
int boosted = w.attack();
